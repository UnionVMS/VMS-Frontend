The search field is used for centering the map on a specific ship.
You can search ships by name in this field and once you have chosen the desired ship in the dropdown the map will center around that ship.
The dropdown is sorted alphabetically and shows the top 10 results.
The search will try to find any ship containing that letter combination.

Example

The search query as will give you ships such as Askungen but also ships like Basto.
Advanced searches:

!

By starting with an exlamation mark the filter will instead try to find any ship not containing the given text.

Example

The filter query !as will give you all ships not shown in the previous example.

/c

By starting your command with /c you can go directly to a specific cordinate.

Example

/c 58.43562 17.45620
